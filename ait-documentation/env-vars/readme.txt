Scripts para crear las variables de entorno del la plataforma con los valores por defecto

***********IMPORTANTE***********
1. Ejecución:
Recuerde ejecutar el script en una consola con permisos de administrador

2. Cifrado y decrifrado de propiedades
Si desea cambiar la clave de cifrado, deberá actualizar las propiedades cifradas presentes en los archivosde configuración.
Puede usar como guia los lineamientos datos por spring cloud:
https://cloud.spring.io/spring-cloud-config/spring-cloud-config.html#_encryption_and_decryption
